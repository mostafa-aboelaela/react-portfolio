# Personal-Portfolio
> This is personal portfolio website.

 
## Built With

- React
- CSS
- VS code
# Getting Started
## Prerequisites:
- Create an account on github

- Install gitbash or use git on the command line.

- Use an IDE of your preference such as VScode.

## Steps to follow:

 - ```
  npm install
  ``` 
 - ```
  npm start
  ```
